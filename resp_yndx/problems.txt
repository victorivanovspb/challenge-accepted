Задачки

    1.  Есть два списка разной длины. В первом содержатся ключи, а во втором — значения.
        Напишите функцию, которая создает из этих ключей и значений словарь. Если какому-то
        ключу не хватило значения, в словаре должно быть значение None. Значения, которым
        не хватило ключей, нужно игнорировать.

            lists_1.py


    2.  Предположим, что у вас есть access.log веб-сервера. Как с помощью стандартных
        консольных средств найти десять IP-адресов, от которых было больше всего запросов?

            p2_shell.txt


        Как то же самое сделать с помощью скрипта на Python?

            p2_logparser.py


    3.  Есть большая таблица в mysql на 15TB со 150 миллиардами записей. Таблица содержит
        два поля: id и value:

            1 CREATE TABLE `data` (
            2 `id` bigint NOT NULL AUTO_INCREMENT,
            3 `value` blob,
            4 PRIMARY KEY (`id`)
            5 )

        В какой-то момент было решено перенести эти данные в другое хранилище, для каждой
        записи из старой таблицы применить некоторое преобразование и поместить в другую.
        
        Дополнительные данные: записи независимы, в новом хранилище возможны batch-записи,
        mysql-сервер один. 

        Необходимо предусмотреть возможные проблемы c сетью и c базами данных, а также
        обеспечить правильное восстановление после сбоя.
